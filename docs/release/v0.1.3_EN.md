# Release v0.1.3

**Release Date**: 2025-12-07

## New Features

### vscode-devgen
- Added LSP integration for `@method` annotation with gopls support for method navigation and completion
- Improved method lookup accuracy using direct regex parsing of method signatures

## Bug Fixes

### vscode-devgen
- Fixed braceCount logic error in `parseTypes` function that caused type parsing to return empty array
- Fixed annotation position calculation using trimmed line causing index offset issues

### Makefile
- Fixed `VSCODE_VERSION` value when HEAD has no tag

## Documentation

- Added README documentation for vscode-devgen and vscgen
- Added devgen.toml configuration examples
