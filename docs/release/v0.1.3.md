# Release v0.1.3

**发布日期**: 2025-12-07

## 新特性

### vscode-devgen
- 新增 `@method` 注解的 LSP 集成，支持 gopls 方法跳转和补全
- 使用正则表达式直接解析方法签名，提升方法查找的准确性

## Bug 修复

### vscode-devgen
- 修复 `parseTypes` 函数中大括号计数逻辑错误，解决类型解析返回空数组的问题
- 修复注解位置计算使用裁剪后的行导致索引偏移的问题

### Makefile
- 修复 `VSCODE_VERSION` 在 HEAD 没有 tag 时的取值问题

## 文档

- 新增 vscode-devgen 和 vscgen 的 README 文档
- 添加 devgen.toml 配置示例
