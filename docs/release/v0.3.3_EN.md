### AI Rules Adapter System

Implemented a brand new AI Rules Adapter system that generates customized rule documentation for multiple AI coding assistants.

**Supported AI Assistants:**
- **Kiro** - Outputs to `.kiro/steering/*.md`
- **CodeBuddy** - Outputs to `.codebuddy/rules/*.mdc`
- **Cursor** - Outputs to `.cursor/rules/*.mdc`

**New Commands:**

```bash
# List supported AI agents
devgen rules --list-agents

# Preview rules content (without writing files)
devgen rules --agent kiro

# Generate rules files
devgen rules --agent kiro -w
devgen rules --agent codebuddy -w
devgen rules --agent cursor -w
```
