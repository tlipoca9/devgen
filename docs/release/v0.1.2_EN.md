# Release v0.1.2

**Release Date**: 2025-12-07

## New Features

### validategen
- Validation rules are now sorted by fixed priority order, ensuring consistent and predictable generated code

### genkit
- Added `IgnoreGeneratedFiles` option to automatically ignore syntax errors in generated files by checking for `// Code generated` header
- Resolved the issue where generated files with syntax errors could not be regenerated

## Improvements

### enumgen & validategen
- Added panic handling for exceptional cases with clearer error messages
- Panic messages now include filename and line number for easier debugging
