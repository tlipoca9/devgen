# v0.3.6 Release Notes

## New Features

### 1. Kubernetes Resource Validation Annotations

Four new Kubernetes-related validation annotations for validating resource specifications and DNS names:

#### @dns1123_label - DNS Label Validation

Validates strings conform to RFC 1123 DNS label specification, suitable for Kubernetes object naming.

```go
// validategen:@validate
type KubernetesObject struct {
    // validategen:@required
    // validategen:@dns1123_label
    Namespace string  // "default", "kube-system" ✓

    // validategen:@required
    // validategen:@dns1123_label
    PodName string  // "my-pod-123" ✓, "Pod" ✗, "-invalid" ✗
}
```

**DNS Label Rules:**
- Only lowercase letters, digits, and hyphens allowed
- Must start and end with alphanumeric character
- Maximum 63 characters per label

#### @cpu - CPU Resource Validation

Validates Kubernetes CPU resource quantity format.

```go
// validategen:@validate
type PodSpec struct {
    // validategen:@required
    // validategen:@cpu
    CPURequest string  // "500m", "1", "2" ✓
}
```

**Supported Formats:** Millicores (`500m`), cores (`1`, `0.5`), scientific notation (`1e3m`)

#### @memory - Memory Resource Validation

Validates Kubernetes memory resource quantity format.

```go
// validategen:@validate
type PodSpec struct {
    // validategen:@required
    // validategen:@memory
    MemoryRequest string  // "128Mi", "1Gi" ✓
}
```

**Supported Formats:** Binary units (`128Mi`, `1Gi`), decimal units (`128M`, `1G`), bytes

#### @disk - Disk Resource Validation

Validates Kubernetes storage resource quantity format.

```go
// validategen:@validate
type PersistentVolume struct {
    // validategen:@required
    // validategen:@disk
    StorageRequest string  // "10Gi", "100Gi" ✓
}
```

### 2. String Enum Alias Type Support

`@oneof_enum` now supports string-based enum alias types:

```go
// enumgen:@enum(string)
type Priority string

const (
    PriorityLow    Priority = "low"
    PriorityMedium Priority = "medium"
    PriorityHigh   Priority = "high"
)

// validategen:@validate
type Task struct {
    // validategen:@oneof_enum(Priority)
    Priority Priority  // Automatically validates against valid Priority values
}
```
